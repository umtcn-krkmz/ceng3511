kakuros.py

this python3 csp code solves 3x3 kakuro puzzles. prints the output if there is a solution and writes the output to the kakuro_output.txt
	for interpretion:
		python3 kakuros.py kakuro_input.txt

futoshiki.py

this python3 csp code solves 4x4 futoshiki puzzles. prints the output if there is a solution and writes the output to the futoshiki_output.txt
	for interpretion:
		python3 futoshiki.py futoshiki_input.txt

NOTE: PYTHON SHOULD BE 64-BIT IN ORDER TO USE THE ORTOOLS LIBRARY
